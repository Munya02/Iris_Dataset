#  Iris Data Visualization

##  Project Overview

This project focuses on visualizing the Iris dataset to understand patterns and relationships between features.
Data visualization helps turn raw data into meaningful insights using graphs and charts.

---

##  Objective

* Create different types of visualizations
* Understand feature distributions
* Compare relationships between variables

---

## Dataset

* **Name:** Iris Dataset
* **Source:** Seaborn built-in dataset
* **Features:**

  * sepal_length
  * sepal_width
  * petal_length
  * petal_width
  * species

---

## Tools & Technologies

* Python
* pandas
* matplotlib
* seaborn
* Jupyter Notebook (google colab)

---

## 📊 Visualizations Created

### 1. Bar Chart

* Displayed the count of each species
* Showed that the dataset is evenly distributed

---

### 2. Histogram

* Visualized the distribution of sepal length
* Helped understand how values are spread

---

### 3. Scatter Plot

* Compared sepal length and sepal width
* Showed relationships between features

---

### 4. Enhanced Scatter Plot

* Added color (hue) for species
* Clearly showed clustering of different flower types

---

## Key Insights

* All species are equally represented in the dataset
* Sepal length values are mostly centered around a specific range
* Different species form visible clusters when plotted
